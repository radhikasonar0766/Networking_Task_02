3. Why is DNS important before communication begins?

DNS (Domain Name System) is important because it translates human-readable domain names (such as google.com) into IP addresses that computers use to communicate.

Without DNS, users would need to remember and enter IP addresses instead of website names. DNS acts like the internet's phonebook, allowing devices to locate and connect to the correct server before communication starts.
